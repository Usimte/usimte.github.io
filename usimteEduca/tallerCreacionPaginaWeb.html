Taller de creación de tu primera página web

Descripción

La frase que describe este curso es: Cualquiera puede desarrollar en la web!
Este taller busca enseñar los conceptos y tecnologias para la creacion de páginas web básicas. Se estudiaran los lenguajes HTML, CSS y JavaScript y su trabajo conjunto para la construcción de sitios web.
Se contara con 4 sesiones presenciales y la ayuda de varios monitores.

Conocimientos previos

No se requiere ningún conocimiento, solo ganas de aprender y una pisca de creatividad.

Resultados al final del taller

Al final los asistentes abrán construido una página que tendra un buen diseño, animaciones y adaptabilidad para dispositivos móviles.
Adicionalmente se darán las bases para la realización de uno de los siguientes proyectos:

  • Una aplicacion que eso los mapas de google con datos de la universidad.
  • Aplicación que usa algunos de los plug-ins, widgets y efectos visuales de la libreria JQuery UI .
  • Un juego desarrollado en JavaScript para profundizar los conceptos de programación.

Tecnologías utilizadas


HTML5: Se encarga de estructurar el contenido.
CSS3: Da la apariencia visual a la página.
JavaScript: A través de este se maneja la lógica de la aplicación.
