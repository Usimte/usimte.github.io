Taller de comandos básicos de linux

Descripción

Este taller pretende dar una introducción a los comandos básicos de consola. Para que el interesado aprenda a desenvolverse de manera autónoma dentro del sistema usando únicamente la terminal de comandos.

Metodología

Este taller se puede realizar de forma autónoma. La metodología consiste en presentar un caso de estudio que plantea una situación que se debe resolver haciendo uso de la terminal de comandos. En la página se presenta una posible solución

Material necesario

Tener instalada una versión actual de GNU Linux. Los ejemplos del taller se realizaran sobre Ubuntu, pero se puede usar casi cualquier versión de GNU Linux pues la diferencia es mínima entre distribuciones.

¿Quién deberia realizar este taller?

Este taller esta orientado a personas que se estan adentrando en el uso del sistema operativo GNU/Linux y desean aprender a realizar acciones a través de consola.

Caso de estudio N° 1

Utilizando unicamente la consola de comandos Buhoberto necesita crear un directorio llamado "animales" dentro del directorio "Documentos" y dentro de este crear 3 archivos llamados "gato", "tigre" y "pantera"

Tips: Dentro de la jerga de los usuarios de linux no se suele usar la palabra "carpeta" siempre se hace referencia al termino "directorio".
